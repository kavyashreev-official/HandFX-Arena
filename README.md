# HandFX Arena

A fun hand-tracking web app that lets you unleash different visual powers using your hands through your webcam!

## How to Play

1. Open `index.html` in your browser.
2. Allow webcam access when prompted.
3. Show your hands to the camera.
4. Press the number keys to switch between powers.
5. Move your hands to activate the effects.

## Controls

Press these keys on your keyboard to change powers:

### `1` : Fire & Ice

* Left hand controls **Fire**
* Right hand controls **Ice**
* Painting starts after a **10-second delay**

### `2` : Lightning

* Summon **lightning bolts** with hand movement

### `3` : Red & Blue

* Left hand controls **Red**
* Right hand controls **Blue**
* Painting is **disabled** in this mode

## Notes

* Works best in a well-lit environment.
* Make sure both hands are visible to the camera.
* Some visual effects were optimized for smoother performance.
* Built for fun experimentation with **hand tracking + visual powers**.
